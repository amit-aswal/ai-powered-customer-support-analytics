# Project Summary for Recruiters

## Project Name

AI-Powered Customer Support Analytics Platform

---

## One-Line Summary

An end-to-end analytics and BI project that converts raw customer support tickets into clean data, SQL models, Power BI dashboards, NLP insights, and executive recommendations.

---

## Business Use Case

Customer support teams handle thousands of tickets every month. This project helps support leaders understand ticket workload, SLA performance, queue bottlenecks, agent workload, recurring customer issues, customer sentiment, and urgency risk.

---

## Key Features

- Data ingestion from Hugging Face
- Raw data profiling
- Feature engineering
- Data validation
- SQL star schema using DuckDB
- Business KPI reports
- EDA and business insights
- Power BI dashboard
- Sentiment analysis
- Urgency scoring
- Keyword extraction
- AI-style executive summary
- Business recommendations

---

## Dashboard Pages

The Power BI dashboard contains:

1. Executive Overview
2. Queue Performance
3. SLA Performance
4. Agent Workload
5. Ticket Category Analysis
6. Monthly Trend Analysis
