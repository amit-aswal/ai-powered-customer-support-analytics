# Resume Bullets

## Project Title

AI-Powered Customer Support Analytics Platform

---

## Resume Bullet Options

- Built an end-to-end AI-powered customer support analytics platform using Python, SQL, DuckDB, Power BI, and NLP to analyze 61K+ support tickets.

- Designed bronze, silver, and gold data layers for raw ingestion, cleaned feature engineering, and analytics-ready reporting.

- Created automated data profiling reports covering missing values, duplicates, categorical distributions, text lengths, and business data gaps.

- Engineered operational support fields such as ticket status, created date, resolved date, resolution time, assigned agent, SLA status, and ticket category using documented simulation logic.

- Built a DuckDB star schema with one fact table and multiple dimension tables to support SQL-based analytics and Power BI reporting.

- Created SQL KPI reports for ticket volume, SLA compliance, queue workload, agent performance, ticket categories, monthly trends, and SLA breach drivers.

- Developed a 6-page Power BI dashboard covering executive overview, queue performance, SLA performance, agent workload, ticket category analysis, and monthly trends.

- Implemented NLP-based sentiment analysis, urgency scoring, keyword extraction, and negative ticket identification to uncover customer experience risks.

- Generated AI-style executive summaries and business recommendations by combining KPI, EDA, dashboard, and NLP insights.

- Documented the complete project workflow with reproducible scripts, validation reports, business impact reports, and GitHub-ready documentation.
