# Phase 6 Power BI Dashboard Report

## Objective

Phase 6 builds the Power BI dashboard for the AI-Powered Customer Support Analytics Platform.

The dashboard converts the cleaned, modeled, and dashboard-ready support ticket data into business visuals for leadership and operations teams.

---

## Power BI File

The Power BI dashboard file is saved at:

`dashboards/powerbi/customer_support_analytics_dashboard.pbix`

---

## Dashboard Pages Created

The dashboard contains 6 pages:

1. Executive Overview
2. Queue Performance
3. SLA Performance
4. Agent Workload
5. Ticket Category Analysis
6. Monthly Trend Analysis

---

## Business Value

The Power BI dashboard helps support leaders monitor:

- Ticket volume
- Open and closed tickets
- SLA compliance
- SLA breach risk
- Queue workload
- Agent workload
- Ticket category trends
- Monthly ticket movement
- Resolution time performance

## Important Note

Some operational fields used in the dashboard, such as ticket status, created date, resolved date, agent name, resolution time, and SLA status, were simulated in Phase 2 because the original dataset does not include real lifecycle fields.

This is documented clearly for transparency.
